# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/pesari/bot.git
cd bot
chmod +x bot.sh
./bot.sh install
./bot.sh 
# Enter a phone number & confirmation code.
```
### One command
To install everything in one command, use:
```sh
cd $HOME
cd bot
./bot.sh
```
